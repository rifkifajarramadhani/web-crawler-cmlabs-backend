Figure 5. cmlabs SEO Tools JSON-LD Schema Generator   Website Responsive dan Mobile-friendly Responsive web/Responsive Web Design is a method to a website design system that provides an optimal experience for users on a desktop, tablet, and mobile devices. This aims to provide a convenient user experience in accessing the website on any device because the website will adapt according to the device screen. Almost the same as the definition of a responsive website before, a mobile-friendly website is a website that is designed and optimized to display well on smaller screens (tablets/mobile). Referring to the previous explanation, the website can adapt to various screen sizes of the user’s device. Having a responsive website will make the website mobile friendly and the website will be able to respond optimally when users access it from any device. Thus, it can get a high ranking because Google prefers the website to be mobile-friendly. There are 3 techniques how to make a responsive website, including: Create a responsive website manually by the frontend team, using CSS styling and layout. Create a responsive website using a CMS (Content Management Platform) platform like WordPress, Joomla, Drupal, etc. Create responsive websites using website themes/templates.   Multiregional and Multilingual Website A multilingual website is any website that offers content in more than one language. e.g. cmlabs and Sequence websites provide both en and id versions of their app site. A multi-regional website is one that explicitly targets users in different countries. e.g. cmlabs targets users in SEA by adding a multi-region feature to the website URL address (en-MY, en-SG, en-PH, en-VN, etc.).
