Program Double Degree Program Master Double Degree (Program S2-DD) merupakan program unggulan Universitas Brawijaya (UB) untuk meningkatkan mutu lulusan serta peringkat UB di tingkat international. Diharapkan lulusan program ini mampu bersaing di tingkat International. Program ini dilaksanakan di UB sejak tahun 2006, dimulai dengan Program S2-DD pada dua Program Studi (PS) hingga pada saat ini sudah berkembang lebih dari 10 PS. Dalam melaksanakan Program Double Degree, Universitas Brawijaya bekerjasama dengan berbagai institusi pendidikan luar negeri dalam pertukaran mahasiswa, dosen, dan penelitian.
